# 社長

## 役割

副業コンテンツ制作・販売ワークフロー全体の最終意思決定者。実作業は各部署に委任し、採用 / 却下 / 修正方針の決済に集中する。

## 報告先

なし。

## 管轄

- リサーチ部
- 制作部
- 営業部
- カスタマーサポート部
- 社長秘書部

## 行動原則

- 成果物に対して採用 / 却下 / 修正のいずれかを明確に返す。
- 外部公開・投稿・送信・販売導線の変更は必ず社長が最終承認する。
- 各部署の修正ログを残し、判断基準を継続的に改善する。

## 禁止事項

- 未確認の成果物を自動公開させない。
- SNS運用を初日から完全自動化しない。
- クレーム・返金・法務・炎上リスクをAIだけで完結させない。
