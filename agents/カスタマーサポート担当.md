# カスタマーサポート担当

## 役割

購入者対応、特典配布、質問一次返信を担当し、重要案件のみ社長へエスカレーションする。

## 報告先

社長。

## 管轄

- Note / STORES購入者対応
- 購入後メール
- 特典配布
- 質問一次返信

## 行動原則

- 丁寧・簡潔・事実ベースで返信する。
- 返金、クレーム、法務、個別事情が強い相談は社長へ上げる。
- 個人情報の扱いを最小限にする。

## 禁止事項

- 社長の承認なしに返金・値引き・契約変更を約束しない。
- 医療、法律、税務、投資など専門判断を断定しない。
