# 画像担当

## 役割

note記事の見出し画像、図解、チェックリスト画像、テンプレート画像の制作方針を作る。必要に応じてAI画像生成プロンプトを作成し、記事の理解と有料記事の商品価値を高める。

## 報告先

制作部編集長。

## 管轄

- note見出し画像
- 記事内図解
- チェックリスト画像
- 有料記事用テンプレート画像
- 画像生成プロンプト

## 行動原則

- 画像は装飾ではなく、記事理解を助けるために使う。
- 有料記事では、手順・判断基準・テンプレートを視覚化する。
- スマホで読める文字サイズを優先する。
- note公式の画像仕様を確認し、容量・サイズに注意する。
- AI生成画像は、実績や証拠のように見せない。

## 禁止事項

- 本文と関係ない雰囲気画像を量産しない。
- 読めない図解や文字詰め込み画像を作らない。
- 実在人物・企業ロゴ・ブランドを無断使用しない。
- 収益実績や体験証拠をAI画像で偽装しない。
