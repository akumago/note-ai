# Phase 1 チェックリスト

対象期間: Week 1-2  
対象部署: リサーチ部、制作部

## 目的

副業コンテンツ制作のうち、ネタ調査からドラフト作成までをAI会社組織で回せる状態にする。

## 開始条件

- [ ] 競合YouTubeチャンネルを3件決める
- [ ] 調査対象キーワードを5-10件決める
- [ ] Noteで参考にするジャンルを決める
- [ ] 社長の文体サンプルを3-5本用意する
- [ ] 公開前の最終承認者を社長に固定する

## 毎日 19:00 リサーチ部

- [ ] 競合YouTube新着を確認する
- [ ] Xの関連トレンドを確認する
- [ ] Note売れ筋上位10本を確認する
- [ ] 採用候補テーマを3-5件に絞る
- [ ] 根拠と想定アウトプットを添えて社長に提出する

## 社長決済

- [ ] 採用テーマを選ぶ
- [ ] 却下テーマを明示する
- [ ] 制作時に避ける表現を指定する
- [ ] 制作部へ渡す

## 毎日 22:00 制作部

- [ ] Noteドラフト1本を作成する
- [ ] X投稿案5本を作成する
- [ ] YouTubeショート台本3本を作成する
- [ ] 表現リスクを確認する
- [ ] 社長が30分で確認できる形式にまとめる

## 完了条件

- [ ] 1日1セットの制作ドラフトが安定して出る
- [ ] 社長の修正時間が30分以内に収まる
- [ ] 修正ログが次回の制作に反映されている
- [ ] 外部投稿は社長承認後にのみ行われている

## 記録するログ

- 採用されたテーマ
- 却下されたテーマ
- 社長が直したタイトル
- 社長が直した本文表現
- 反応がよかった投稿
- 反応が悪かった投稿
