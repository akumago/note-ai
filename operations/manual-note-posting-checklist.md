# note手動投稿チェックリスト

## 目的

note投稿は自動公開せず、社長が手動で確認して公開する。  
このチェックリストは、公開ミス・画像漏れ・導線漏れ・規約リスクを防ぐために使う。

## 投稿前

- [ ] 投稿パッケージを確認した
- [ ] 採用タイトルを確認した
- [ ] サブタイトルを必要に応じて冒頭に反映した
- [ ] 本文をnoteエディタへ貼り付けた
- [ ] 見出し画像を設定した
- [ ] フロー図PNGを本文に挿入した
- [ ] チェックリストPNGを本文に挿入した
- [ ] 画像がスマホで読めるか確認した
- [ ] 一人称が統一されている
- [ ] `[画像: ...]` の挿入メモを削除した
- [ ] 収益保証・成果保証がない
- [ ] 架空ケースは架空と明示している
- [ ] 仮説は仮説と明示している
- [ ] note内で内容が自己完結している
- [ ] 最後の次アクションが具体的

## ハッシュタグ

初回は以下から5個程度に絞る。

```text
#note
#note運用
#note収益化
#AI活用
#記事作成
#副業
#ブログ
#WordPress
```

## 公開直後

- [ ] 公開URLをコピーした
- [ ] `outputs/metrics/` に公開URLを記録した
- [ ] スマホで表示確認した
- [ ] 画像崩れがないか確認した
- [ ] 誤字を1回だけ確認した
- [ ] X告知を1本だけ投稿した
- [ ] 追加の連投はしない

## 公開24時間後

- [ ] ビュー数を記録した
- [ ] スキ数を記録した
- [ ] コメントを記録した
- [ ] フォロー増を記録した
- [ ] SNS反応を記録した

## 公開72時間後

- [ ] スキ率を確認した
- [ ] コメントや質問の質を確認した
- [ ] 続編候補を確認した
- [ ] タイトル改善の必要性を判断した

## 公開7日後

- [ ] シリーズ化するか判断した
- [ ] 有料テンプレート化するか判断した
- [ ] WordPress保存版にするか判断した
- [ ] 撤退または保留にするか判断した
